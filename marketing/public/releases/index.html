Honest status

What "get the app" means today.

Riot is a prototype in active development. Anything you install from this page is an early build, not a finished or audited product.

Today

No published release yet

The GitHub releases page is the source of truth. If it is empty, there is nothing to install. The home page screenshots are real app screens from the iPhone simulator build — they are not evidence of sync over physical-device radios.

When builds appear

Treat them as prototypes

Do not rely on early builds for anything that must stay secret. Private encrypted groups are not shipped; current public spaces are not confidential. Use established encrypted messengers for sensitive communication. See the privacy stance.

Free, open source, and forkable. If you want to run Riot before a binary ships, the source and build instructions are public — cargo test --workspace --all-features is the canonical verification command.
